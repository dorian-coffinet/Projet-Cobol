Projet de DUT informatique

Création d’un environnement sous Cobol permettant la gestion des prêts d’ouvrages en informatique d’un établissement.

Le projet se découpe de la manière suivante :
Page d’accueil (connexion)-> Menu principal -> Adhérents (Consulter fiche, Ajouter, Modifier, Supprime, Calcul Amende)
											-> Livres (Rechercher, Ajouter, Modifier, Supprimer)
											-> Prêts (Consultation, Modifier, Supprimer, Enregistrer un prêt/retour)
											-> Administration *Visible uniquement si l’utilisateur est connecté en administrateur* 
											   (Ajouter , Modifier, Supprimer un utilisateur, Consulter historique)
